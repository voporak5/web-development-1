Recommended Software


IDE (Integrated Development Environment)

  • Visual Studio Code (Windows/Mac)
  • Brackets (Windows/Mac)
  • Notepad++ (Windows)
  • TextEdit (Mac)
  • Notepad (Windows)


Visual Studio Code

Probably the easiest thing you'll find installed on the machines at school. Very common especially since it isn't exclusively for web development. Don't confuse this with the standard Visual Studio which you can use for web development, but Visual Studio Code is much more friendly.

Brackets

I highly recommend this one because the autofill is very smart and it has other cool features like showing you the color in a preview as you're typing the color out.

Notepad++

This gets the job done and lets you focus on typing out your HTML without distracting you from other windows, editor features, etc. because it's just a better version of Notepad intended for programming. Also it also has intellisense if you write PHP. Brackets does this too but you have to set it up.

TextEdit

Mac version of Notepad so definitely not ideal. It also tries to save your files as rich text files (.rtf) which is just a text file (.txt) with the ability to handle formatting similar to a word doc.

Notepad

Plain text editor on Windows that definitely will allow you to get the job done but you need to know what you are doing since there are no autofill features. Be sure you save your files as .html

Github


Github is what we call a repository service. You can think of a repository service similar to Dropbox, Google Drive, OneDrive, etc. except the primary intent is for storing code. You can actually store whatever files you want, but there are special functions developed around the assumption you're using it to store code. Github provides other features that this class is in no way meant to go over. An additional feature that Github provides is the ability to host a website that you've been developing. The functionality of the website can be pretty limited, but for this class it provides everything we care about. There are other services such as Bitbucket that provide the same or similar features, but Github is the most common.
