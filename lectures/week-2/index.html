TextEdit

Mac version of Notepad so definitely not ideal. It also tries to save your files as rich text files (.rtf) which is just a text file (.txt) with the ability to handle formatting similar to a word doc.

Notepad

Plain text editor on Windows that definitely will allow you to get the job done but you need to know what you are doing since there are no autofill features. Be sure you save your files as .html

Anatomy of an HTML tag


In order to begin building pages you have to get familiar with tags.
Tags are a way for you to define what you want to show on a web page.
You start with a < sign, the name of the tag, and a > sign. This is called the opening tag.

Then you enter your content.

Then you enter the a closing tag which is just like the opening tag but with a / like the below example. 
<h1>
    Hello World
</h1>
Not every tag requires a closing tag. The meta tag for instance just configure your
web page and does not apply any effect to content.
