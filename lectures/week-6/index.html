What Is Bootstrap And Why Are We Using It?

Before we answer that question let's take a look at this and answer the following question...

How long would it take you to put together a layout that resembled this page?

Inspect the page and look and see all of the sections there are. Change the size of the window as well to really see how complicated the web page is.

Getting the margins and padding correct as well as making the content scale up and down really nicely would be a challenge.

This is where Bootstrap comes into play! Bootstrap is a tool that makes putting together a page like the one we just looked at a lot easier.

Simplifies the process for building a layout for your web page in addition to a lot of other cool things. This entire site is actually built using Bootstrap. Bootstrap provides many CSS classes that you can implement on your tags to improve their functionallity or make them do cool things. One example is that most pages have some sort of margins on the left and right so the content is not tight against the edges of the display. You can apply a predefined class called "container" to the body to get nice margins that work for both Desktop and Mobile. 

<body class="container">

</body>

Let's look at Bootstrap


Adding Bootstrap is easy. We just add the CSS and JS libraries (which you may not even use the JS libraries depending on what you actually use Bootstrap for) 

<!doctype html>

  </head>
  <body>
    <h1>Hello, world!</h1>

  </body>
</html>

In order to maximize Bootstrap's potential you'll need to read the website to learn everything it can do however, here are some sections that are really useful.

Containers

As I mentioned you can declare a container class on an element to help size everything in it.

Columns

Bootstrap makes creating <div> elements that are sized to your liking very easy. Essentially Boostrap will allow you to divide the width of the container up to 12 columns.


This Div is 3 Columns Wide

This Div is 6 Columns Wide

This Div is 3 Columns Wide


<div class="row">
    <div class="row col-3">
        <h4>This Div is 3 Columns Wide</h4>
    </div>
    <div class="row col-6">
        <h4>This Div is 6 Columns Wide</h4>
    </div>
    <div class="row col-3">
        <h4>This Div is 3 Columns Wide</h4>
    </div>
</div>

I've created a <div> that I want to put 3 columns worth of content inside. The <div> takes up the full width of the parent container (the body) and 2 of the columns take up 3 columns each or rather 25% and 1 column (the middle) takes up 6 columns or rather 50% of the available space.

If the sum of the columns that the elements are taking up exceeds 12 then the excess elements will be pushed on to a new line like the below example.

This Div is 6 Columns Wide

This Div is 4 Columns Wide

        <h4>This Div is 6 Columns Wide</h4>
    </div>
    <div class="row col-4">
        <h4>This Div is 4 Columns Wide</h4>
    </div>
</div>
