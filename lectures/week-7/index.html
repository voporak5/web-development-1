What is Javascript and How Does It Relate To Web Pages?

Javascript is a way for you to add logic to your website. It's a programming language (although some would argue you aren't actually programming) primarily used for web development. It has other uses as well, but nothing you care about for this class.

In terms of the relation with HTML and CSS, HTML is for defining what to put on a page, CSS is for defining how the content will be presented, and JS can add logic to make it smart and more intuitive to users.

You can handle logging someone in, build a game, or throw popups asking the user to confirm their choice before submitting as a few examples of the limitless potential.

Maybe you want to make a calculator that adds two numbers together?

Basics

One of the most popular things to use programming for is math. We could very easily write a line of code to print the output of 235/6 like this 

    document.write('<p>' + (235/6) + '</p>');

Which is...

But that's not very useful because the input are more than likely to change so what we need to do to help us is use variables

Variables are things that let us store a value that we can easily use throughout the program. If you think back to math class (1 + x = 4) x is a variable that could represent anything. In this particular example we definitely know the value is 3. In math class we usually try to figure out what the value is of the variable, but in programming we allow the code to generate the result based on whatever the variable changes into.

Creating a variable is easy. We use the keyword var to say "I want to create a variable" and then you specify what the name of the variable will be. It can be single letters or full words. It can also have numbers in it but it cannot have special characters.

You can also use the keyword let as an alternative to var however, the difference between the 2 is hard to explain. Additionally you can use const which simply meens the value of the variable cannot be changed ever which can be useful when working with teams so team members don't accidentally change the value of variables that shouldn't change.

When creating a variable you can specify a value immediately or change it later if it wasn't created using const. 

    var x;
    let firstNumber;
    const PI = 3.14;
    
    x = 100;
    firstNumber = 50;

There are 3 basic types of values we can assign to a variable which are numbers, words, and booleans (true/false).

Whole numbers that do not have decimal values (including 1.00) are called integers.

Decimal value numbers such as 1.00 or 1.2111234 are called floats.

Words that have more than 1 letter are called strings.

Single letters are called characters (which more than 1 character creates a string). 

    var name = "Cameron";
    var lastInitial = 'C';
    var age = 31;
    var isMarried = true;
    var numberOfBathrooms = 2.5;

Something to point out is that whenever we end a statement (a single piece of code that we want to execute) we should add a semicolon ; to the end. This is the equivalent of adding a period in grammar. Javascript will still work without the semicolon while other programming languages will fail so it's good to just get into a habit of using semicolons;

Plus they making reading the code easier especially when the code is compressed to reduce all white space like this. 

    var name = "Cameron"; var lastInitial = 'C'; var age = 31; var isMarried = true; var numberOfBathrooms = 2.5;

We can easily inject the variable into Javascript like this 

    var name = "Cameron";
    document.write('<h1>' + name + '</h1>');

This isn't very usuable though because each line executes one time and we might as well just write everything inside of the parentheses, but if we try referencing the variable multiple times in our webpage. That way if the value needs to change we only have to change it in 1 place.

What would also be very useful too is if there was a way for the page to update if the value of the variable changes.

To accomplish this we need to create something called a function. A function is a definition of code that will execute the lines of code inside the definition wherever the function is referenced. 

    var name = "Cameron";
    
    var printName = function(){
        document.write('<h1>' + name + '</h1>');
    };

    printName();
    
    name = "Larry";
    
    printName();

Arrays

What if you wanted to create a bunch of names? You could make a bunch of variables called name1, name2, name3, etc. This would be time consuming though and if more names needed to be stored then you would have to edit the code again.

A solution to this is to use Arrays which are lists and creating one is easy. Just declare a variable like before but set the value as a pair of brackets.

This will create an empty list but, you can also create a list that has some data already in it but using curly braces.

You can add content to the array by referencing the variable and adding .push() to the end with whatever you want to put in the array being placed inside the parentheses, but this isn't anything we need for right now.

If you want to access items out of the array you add brackets after the variable and put the index of the item you want to access inside. FYI... Indexes start at 0 instead of 1

You can also get the size of the array by appending .length to the variable 

    var names = [];
    var cars = ["nissan","volvo","tesla"];
    
    names.push("Cameron");
    names.push("Larry");
    
    //Write nissan
    document.write('<h1>' + cars[0] + '</h1>');
    
    //Write volvo
    document.write('<h1>' + cars[1] + '</h1>');
    
    //Write tesla
    document.write('<h1>' + cars[2] + '</h1>');
    
    //Should print out "There are: 3 cars"
    document.write('<h1> There are: ' + cars.length + ' cars</h1>');

If you wanted to get a random value out of the list you could store the random number into a variable and use that when accessing the array. 

    var cars = ["nissan","volvo","tesla"];
    var randomIndex = Math.floor(Math.random() * cars.length);
    
    document.write('<h1>' + cars[randomIndex] + '</h1>');

Math.random() returns a random float between 0 and 1 so we have to multiply that by the max number of values there can be. We then round the number down so that it will return 0, 1, or 2 when we say 3.
