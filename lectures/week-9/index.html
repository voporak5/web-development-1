Loops

Consider a scenario in which you have to build a website that populates an ever growing list of content on a webpage such as your social media wall (Facebook, Instagram, X, etc.) with posts from other user. It's not realistic to manually type all of the HTML or make all of the JS function calls because you'd have to go change the content every time and the content might be different for each person who visits the site.

The realistic solution is to use a programming loop, or in other words some lines of code that will repeatedly run until a set of conditions are met that will break the loop.

Conditions can be as simple as run 10 times or as dynamic as keep running until there is no more new content to show to the user.


Types of Loops

There are 3 main types of loops that you will encounter
  • for loops
  • while loops
  • do while loops

For Loops

To begin writing the for loop you must use the keyword "for" followed by a set of parentheses. After the parentheses ( ) you need a set of curlybraces { }.

The code you want to execute each iteration of the loop must go inside the curlybraces.

The conditions for how many times the loop must execute go inside the parentheses.

With the for loop we must specify 3 parts in the conditions: a variable that represents a counter, the condition for terminating the loop, and a method for incrementing the counter.

Each of these parts must be separated by a semicolon.

As you can see below I have created a variable called i which is shorthand for index. I then said that the loop can only run if i is less than 10. Finally I stated that at the end of the loop we will increment i by 1. Specifying++ is shorthand for saying add 1. I could also say i += 1 or if I wanted it to increase more I could say i += 2.

You don't have to call your variable i but you will see i used very often. 

for(var i = 0; i < 10; i++){
    document.write('<h4>hi</h4>');
    document.write('<h4> this is loop: ' + i + '</h4>');
}

While Loops

While loops are similar to for loops but the only difference is that they use the keyword "while" and the only thing that goes inside the parentheses are the conditions for terminating the loop. 

var isGameRunning = true;
    
while(isGameRunning == true){
    spawnEnemy();
}

If you wanted to you could use a while loop exactly like a for loop. You would have to create the variable outside of the loop and manage incrementing the value inside of the loop. 

var enemiesSpawned = 0;
    
while(iterations < 10){
    spawnEnemy();
    enemiesSpawned++;
}

Do While Loops

These are very similar to while loops however, it must be said that a for loop and a while loop will not run if the conditions to terminate the loop are already met by the time the loop begins to run because a do while loop will always guarantee 1 iteration of the code executing before checking to see if the conditions to terminate the loop are met.

It also looks slightly different in that the keyword "do" is used but instead of a set of parentheses following it the curly braces are immediately placed and at the end the "while" keyword is added along with the parentheses that hold the conditions for terminating the loop. 

var lives = 3;
    
do{
    dealDamage();
    lives--;
}while(lives > 0)

Document Object Model (DOM)


The Document Object Model was developed as a way to access the underlying structure of a Web document (web page). It's a set of rules implemented by every web browser to allow languages such as Javascript to access the elements on the page and modify their properties such as color and positioning.

Essentially, what you see as a bunch of pictures and pieces of text that are just being placed sequentially from top to bottom are actually a part of a tree of elements that together creates the hierarchy of the web page.


There are several functions that you can use to find elements and modify or add new behaviors to them. Learning them all will take time and practice so it helps to have a goal you are trying to reach such as create a picture that will move to each corner and then doing some research to see what properties are available on an image such as x and y position.
